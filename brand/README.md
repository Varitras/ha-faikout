# Brand assets

`icon.png` (256×256) and `icon@2x.png` (512×512), transparent PNG.

Home Assistant does not read icons from an integration repository — it serves them from
[home-assistant/brands](https://github.com/home-assistant/brands). Until these files are submitted
there as `custom_integrations/faikout/`, Home Assistant shows the default placeholder.

The mark is original artwork for this integration: it deliberately does not use Daikin's or the Faikin
project's branding.
